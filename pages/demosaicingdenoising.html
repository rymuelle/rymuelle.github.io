Raw Refinery

What is Raw Refinery

Raw Refinery is the first open source RAW image denoiser, with the novel goal of additionally improving the aesthetic characteristics of the noise, rather than simply improving the signal to noise ratio. Operating on raw sensor data, it produces a high quality denoised DNG file ready to be edited in your favorite raw editing program. .

The tech behind Raw Refinery

In order to best reconstruct clean images, RawRefinery works on the (relatively) unmodified sensor data, in the form of an ADC readout of each pixel. ach pixel has a color filter over it, commonly in a Bayer array, which is a color filter array (CFA) of red, green, and blue pixels. It is preferable to work on low level data for denoising, but transforming this mono image into an color image is both an additional challenge and opportunity. The machine learning model learns to both "demosaic", that is, transform the mono image into a 3 channel color image, and denoise the the image simultaneously. RawRefineries machine learning model is built on a modified version of the NAFNet U-Net, enabling fast, high quality results that can be computed on your local machine. Custom channel attention modules enable the network to use a prior about the noise levels both to increase it's performance, and give the end user control over the model output, and novel skip connection merging optimally combines high level global information with fine grained local details to preserve textures while reducing noise. The model is trained on the RawNIND dataset.
